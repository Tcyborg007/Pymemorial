Memorial de Cálculo - Viga Biapoiada

1. Dados do Problema

Considere uma viga biapoiada submetida a uma carga uniformemente distribuída. Os dados de entrada são:

\(q = \frac{10}{m} = 10.0 \, \text{kN/m}\)

\(L = 6 = 6.0 \, \text{m}\)

2. Cálculo do Momento Máximo

sabemos que o momento fletor máximo (M_max) em uma viga biapoiada é definido pela fórmula

\[M_{max} = q \times L^2 / 8\]

Substituindo os valores fornecidos:

\(M_{max} = 45 = 45.0\)

\(M_{max} = 45 m · = 45.0 \, \text{kN}\)

3. Cálculo da Tensão

Considere uma seção com módulo de resistência:

\(W = 8.0 \cdot 10^{-9} = 0.002 \, \text{m³}\)

A tensão normal é calculada por:

sabemos que a tensão é dada pela relação momento sobre módulo

\[\sigma = M_{max} / W\]

Substituindo:

\(\sigma = 22500.0 = 22500.0\)

\(\sigma = 22500 = 22500.0 \, \text{kPa}\)

\(\sigma = 22.5 = 22.5 \, \text{MPa}\)

4. Verificação

A tensão de escoamento do aço é:

\(f_{y} = 250 = 250.0 \, \text{MPa}\)

Comparando com a tensão atuante:

\(σ < f_{y}\) (True)

**Conclusão**: Como

\(σ < f_{y}\) (True)

, a viga está **APROVADA**. A razão de utilização é:

\(\eta = \frac{σ}{f_{y}} = 0.09\)

\(\eta = 0.09 = 0.09\)

\(\eta = 0.09 = 0.09\)

Portanto, a viga está trabalhando a apenas **9%** de sua capacidade.
