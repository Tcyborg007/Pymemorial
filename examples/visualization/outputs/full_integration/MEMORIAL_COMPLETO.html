5. CÁLCULOS DE CAPACIDADE

5.1 Índice de Esbeltez

λ = L / rx = 4.00 / 0.136 = 29.40
Referência: NBR 8800:2024 - Item 5.3.2

5.2 Fator de Redução

χ = 0.877 (para λ = 29.40 < 100)

5.3 Capacidade Axial

NRd = χ × Ag × fy / γa
NRd = 0.877 × 137.32 × 345 / 1.1
NRd = 3777 kN
Referência: NBR 8800:2024 - Item 5.3.3

5.4 Capacidade Flexional

MRd = fy × Zx / γa
MRd = 345 × 1635 / 1.1
MRd = 513 kN·m
Referência: NBR 8800:2024 - Item 5.4.2

6. VERIFICAÇÃO

η = √((Md/MRd)² + (Nd/NRd)²)
η = √((150/513)² + (1200/3777)²)
η = 0.432 = 43.2%
✓ APROVADO (η < 1.0)
