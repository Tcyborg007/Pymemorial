Memorial de Cálculo - Viga V1 (Térreo)

1. Dados Gerais

1.1. Geometria da Viga

L1 = 5.0 m

L2 = 6.0 m

L3 = 5.0 m

b_w = 20.0 cm

h = 50.0 cm

cobrimento = 3.0 cm

1.2. Materiais

fck = 30.0 MPa

fyk = 500.0 MPa

gamma_c = 1.4

gamma_s = 1.15

Ecs = 5600 * sqrt(fck)

1.3. Propriedades da Seção

Ac = b_w * h

Ic = b_w * h**3 / 12

d = h - cobrimento - 1.0

2. Cargas Atuantes

2.1. Cargas Permanentes (g)

g_alv = 3.5 kN/m

g_rev = 2.0 kN/m

g_pp = (Ac / 10000) * 25

g_total = g_alv + g_rev + g_pp

2.2. Cargas Acidentais (q)

q_k = 3.0 kN/m

2.3. Carga de Cálculo (qd)

gamma_g = 1.4

gamma_q = 1.4

q_d = gamma_g * g_total + gamma_q * q_k

3. Esforços Solicitantes de Cálculo

3.1. Momentos Fletores (Md)

M1_pos_d = q_d * L1**2 / 14

M1_neg_d = -q_d * L1**2 / 10

M2_pos_d = q_d * L2**2 / 16

M2_neg_d = -q_d * L2**2 / 9

3.2. Forças Cortantes (Vd)

Vd_apoio1 = 0.45 * q_d * L1

Vd_apoio2_esq = 0.60 * q_d * L1

Vd_apoio2_dir = 0.60 * q_d * L2

Vd_max = max(Vd_apoio1, Vd_apoio2_esq, Vd_apoio2_dir)

4. Verificações Normativas (Simplificadas)

4.1. Verificações Geométricas

4.2. Dimensionamento à Flexão (Simplificado)

fcd = fck / gamma_c

fyd = fyk / gamma_s

rho_min = 0.15 / 100

As_min = rho_min * Ac

4.3. Verificação ao Cisalhamento (Simplificado)

alpha_v2 = (1 - fck / 250)

VRd2 = 0.27 * alpha_v2 * fcd * b_w * d

5. Resumo
